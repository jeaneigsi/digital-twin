## Style Prompt

Clean editorial technical — light canvas, sharp typographic hierarchy, minimal color. Swiss-inspired precision without the rigidity. Data and code feel at home alongside prose. Calm, clear, credible.

## What NOT to Do

- No gradients, no glow, no drop shadows
- No dark mode — light canvas only
- No centered-and-floating single elements — anchor to edges or use grids
- No more than 3 font sizes per scene
- No neon, no purple, no cyan
